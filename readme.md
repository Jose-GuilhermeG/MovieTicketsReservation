# CineTickets 
Sistema de gerenciamento de ingressos de filmes, 
criado com o intuito de colocar em prática e reforçar
conhecimentos sobre:
    - Desenvolvimento de software com:
        - java + spring boot (api principal)
        - python para análise dados e alguns microservices 
        - Padrões de projeto e boas práticas
    - Arquitetura de software com:
        - microservices
        - arquitetura em camadas
    - Monitoramento com:
        - logs
        - promethues + grafena
    - Escalabilidade
